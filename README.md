# Timmy Tails - Pet Grooming Appointment System

AI-powered pet grooming appointment booking platform with ML-driven haircut recommendations by breed.

## 🚀 Features

### Frontend (React + Tailwind)
- ✅ Home page with hero section
- ✅ Services showcase
- ✅ About & Contact pages
- ✅ User authentication (Login/Signup)
- ✅ Booking system with breed selection
- ✅ Admin dashboard with analytics
- ✅ ML Trends tab for AI predictions

### Upcoming - Backend
- Express.js REST API
- Python Flask ML microservice
- MongoDB database
- JWT authentication
- Real-time appointment management

## 📋 Tech Stack

**Frontend:**
- React 18
- React Router v6
- Tailwind CSS
- Framer Motion (animations)
- Lucide icons

**Backend (In Progress):**
- Node.js + Express.js
- Python + Flask (ML)
- MongoDB
- scikit-learn (ML models)
